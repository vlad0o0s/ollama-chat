# Устранение неполадок

## Ошибка прокси: ECONNREFUSED

Если вы видите ошибку:
```
Proxy error: Could not proxy request /api/auth/verify from localhost:3000 to http://localhost:5000/.
ECONNREFUSED
```

Это означает, что frontend не может подключиться к backend серверу.

### Решение:

1. **Проверьте, запущен ли backend:**
   ```bash
   # Windows PowerShell
   netstat -ano | findstr :5000
   
   # Должен показать процесс, слушающий на порту 5000
   ```

2. **Если backend не запущен, запустите его:**
   ```bash
   cd backend
   venv\Scripts\activate
   python run.py
   ```

3. **Или используйте скрипт запуска:**
   ```bash
   # Из корневой директории проекта
   start.bat
   ```

4. **Проверьте, что backend отвечает:**
   Откройте в браузере: http://localhost:5000/health
   
   Должен вернуться: `{"status":"ok"}`

5. **Если backend запущен, но все равно ошибка:**
   - Подождите несколько секунд (backend может перезагружаться)
   - Перезапустите frontend (Ctrl+C и `npm start` снова)
   - Проверьте, что порт 5000 не занят другим процессом

### Порядок запуска:

1. Сначала запустите **backend** (порт 5000)
2. Подождите 3-5 секунд
3. Затем запустите **frontend** (порт 3000)

### Альтернативный способ запуска:

Используйте `start.bat` из корневой директории - он запустит оба сервера автоматически.
